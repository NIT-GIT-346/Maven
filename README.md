# Project Documentation

## Overview
This project is a Maven-based Java application.

## Getting Started

### Prerequisites
- JDK 11 or higher
- Maven 3.6 or higher

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/NIT-GIT-346/Maven.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Maven
   ```
3. Install the dependencies:
   ```bash
   mvn install
   ```

## Usage
To run the application, use the following command:
```bash
mvn exec:java
```
